**Type-specific content requirements:**

**haftpflicht.md:**
- Benchmark: Deckungssumme >= EUR 5M (pauschal for Personen-, Sach-, Vermogensschaden)
- Common components: Mietsachschaden (often excluded — flag this), Schluesselverluest, Gefalligkeitsschaden, Forderungsausfalldeckung
- Source: section 823 BGB (general liability), VVG

**hausrat.md:**
- Benchmark: >= EUR 650/m2 Wohnflache with Unterversicherungsverzicht clause
- Common components: Einbruchdiebstahl, Feuer, Leitungswasser, Sturm/Hagel, Fahrrad-Zusatz (up to EUR 3,000-5,000), Elementarschaden (optional but increasingly important)
- Note: `coverage_amount` is sum-based — total Versicherungssumme in EUR

**kfz.md:**
- Three coverage levels: Haftpflicht (mandatory per section 1 PflVG), Teilkasko, Vollkasko
- Document SF-Klasse (Schadenfreiheitsklasse) impact, Typklasse, Regionalklasse
- Benchmark: Haftpflicht mandatory, Vollkasko recommended for financed vehicles
- `coverage_components` should capture which levels are active: ["Haftpflicht", "Teilkasko"] or ["Haftpflicht", "Teilkasko", "Vollkasko"]
- One policy entry per contract, NOT per coverage level (per research pitfall 3)

**rechtsschutz.md:**
- Common components: Berufsrechtsschutz, Verkehrsrechtsschutz, Mietrechtsschutz, Privatrechtsschutz
- Wartezeit: typically 3 months
- `coverage_amount` is null — service-based, not sum-based
- Add `coverage_type: service_based` note for portfolio agent (per research pitfall 5)

**zahnzusatz.md:**
- Benchmark: >= 80% Zahnersatz reimbursement; ideally no Wartezeit
- Context: GKV pays only 60% base Festzuschuss for dental prosthetics
- Common components: Zahnersatz, Zahnbehandlung, Kieferorthopadie, Professionelle Zahnreinigung, Implantate
- Note Staffelung (graduated coverage in first years, e.g., year 1: max EUR 1,000, year 2: EUR 2,000)

**risikoleben.md:**
- Benchmark: Versicherungssumme = 3-5x annual gross income if dependents exist
- Health underwriting similar to PKV (Gesundheitsfragen)
- Common components: Todesfallschutz, Nachversicherungsgarantie, Beitragsbefreiung bei BU
- Note: declining term (linear fallende Versicherungssumme) vs level term

Each reference doc should be 80-120 lines. Use WebSearch to verify key benchmarks against GDV/Verbraucherzentrale if uncertain. Flag training-data-sourced benchmarks with `[VERIFY]` if confidence is low.

**Type-specific content requirements:**

**reise.md:**
- Bundle types: Auslandskrankenversicherung, Reiserucktrittsversicherung, Reisegepeckversicherung, Reisehaftpflicht
- Coverage lapses: many policies cap at 42-56 days per trip — flag this
- `coverage_amount` is null — service-based

- Benchmark: Must include Krankenrucktransport (medical repatriation)
- Source: BaFin, GDV, Stiftung Warentest Reiseversicherung tests

**fahrrad.md:**
- Coverage: Diebstahl, Vandalismus, Unfallschaden, Verschleiss (some policies)
- E-bike specific: Akkuschaden, Elektronikschaden riders
- Note: Often available as Hausrat add-on (Fahrrad-Zusatz) — flag overlap with Hausrat
- Benchmark: Neuwertentschadigung (replacement cost), not Zeitwert
- `coverage_amount` is sum-based — insured value of bicycle in EUR
- Source: GDV, Verbraucherzentrale

**kfz-schutzbrief.md:**
- Coverage: Pannenhilfe (breakdown), Abschleppen (towing), Mietwagen (rental car), EU-wide assistance
- Note: Often included in ADAC membership or as Kfz policy add-on — flag overlap
- `coverage_amount` is null — service-based
- Add `coverage_type: service_based` note
- Benchmark: EU-wide coverage, 24h hotline, towing up to nearest workshop
- Source: ADAC, GDV

**mietkaution.md:**
- Replaces cash Mietkaution (rental deposit, max 3x net cold rent per section 551 BGB)
- Requires credit check (Bonitat/Schufa)
- Annual premium typically 3-5% of deposit amount
- `coverage_amount` is sum-based — the guaranteed deposit amount in EUR
- Alternative: Bankbuergschaft (bank guarantee)
- Source: section 551 BGB, Verbraucherzentrale
- Benchmark: Coverage must equal full deposit amount (3x Nettokaltmiete)

Each doc: 60-100 lines. Same YAML frontmatter pattern. Same Field Extraction Schema table with profile.json field names.
