**`<objective>` section:**
Deliver a personalized PKV vs GKV comparison by orchestrating eligibility checks, a health questionnaire, and two specialist agents. The command:
1. Checks PKV eligibility against JAEG threshold (per D-01, D-02)
2. Emits age-55 lock-in warning when applicable (per D-03)
3. Detects expat status and shows Anwartschaft/portability guidance (per D-06, D-07)
4. Collects health questionnaire via single multiSelect (per D-04, D-05)
5. Spawns calc + research agents in parallel (per D-08, D-09)
6. Emits legal disclaimer BEFORE any advisory content (per D-10, matching insights.md pattern)
7. Synthesizes unified comparison with recommendation

This command writes NO files. All output is conversational advisory text.

**`<process>` section — 7 phases per D-01:**

**Phase 1: Validation and Eligibility Gate (ELIG-01)**
- Check profile exists: `[ -f .finyx/profile.json ]` — error message directing to `/finyx:profile` if missing
- Extract from profile: `countries.germany.gross_income`, `identity.family_status`, `identity.children`, `identity.cross_border`, `identity.income_countries`, `countries.germany.marginal_rate`
- Check Germany is active: `countries.germany.tax_class != null` — if not, error "Health insurance advisor requires German income data"
- Staleness check: Compare `tax_year` in health-insurance.md frontmatter against current year, warn if mismatch (same pattern as tax.md Phase 1)
- **Eligibility gate:** Compare `countries.germany.gross_income` against JAEG threshold (€77,400 for 2025). Read the exact JAEG value from the loaded health-insurance.md Section 4.1 — do NOT hardcode it.
  - If income < JAEG: emit GKV-mandatory message per D-02:
    ```
    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
     FINYX ► INSURANCE: GKV MANDATORY
    ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    Your gross income (€XX,XXX) is below the Versicherungspflichtgrenze
    (JAEG: €77,400/year). PKV is not available.

    Your estimated GKV cost:
    [Spawn calc agent with gkv_only=true flag, show GKV breakdown only]
    ```
    Then STOP — do NOT proceed to questionnaire or full comparison.
  - If income >= JAEG OR employment type is self-employed (self-employed are exempt from JAEG): proceed to Phase 2.
  - Note: Self-employed persons can choose PKV regardless of income (per health-insurance.md Section 4.2). Detect via profile employment data or ask inline if not in profile.

**Phase 2: Warnings and Expat Detection (EDGE-01, EDGE-02)**
- **Age-55 lock-in warning (per D-03):** Calculate user age from profile (or ask if not present). If age >= 50 and PKV is viable, emit a prominent WARNING banner BEFORE the questionnaire:
  ```
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   ⚠️  AGE-55 LOCK-IN WARNING (§6 Abs. 3a SGB V)
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  You are [age] years old. Under §6 Abs. 3a SGB V, if you switch
  to PKV and remain insured for 10+ years past age 55, returning
  to GKV becomes near-impossible — even if your income drops below
  the JAEG threshold.

  This is effectively an IRREVERSIBLE decision at your age.
  Consider this carefully before proceeding.
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ```
  Use the exact text from health-insurance.md Section 6.3 for the legal basis.

- **Expat detection (per D-06):** Check `identity.cross_border === true || identity.income_countries.length > 1`. If both signals are absent, ask the user inline via AskUserQuestion: "Are you or have you been a resident outside Germany? (yes/no)". If expat confirmed, set `show_expat = true` for Phase 5.

**Phase 3: Health Questionnaire (per D-04, D-05)**
- Use AskUserQuestion with multiSelect to collect all 15 binary health flags in a single round-trip per D-04.
- Prefix each flag with category code: `[CV]` cardiovascular, `[MET]` metabolic, `[MSK]` musculoskeletal, `[MH]` mental health, `[OTH]` other.
- The 15 flags (derive from calc agent Phase 3 risk tier mapping):
  - [CV] High blood pressure (diagnosed/treated)
  - [CV] Heart condition or cardiac event
  - [CV] Elevated cholesterol (treated)
  - [MET] Diabetes (Type 1 or Type 2)
  - [MET] Thyroid condition (treated)
  - [MET] BMI above 30
  - [MSK] Chronic back condition
  - [MSK] Joint condition (diagnosed)
  - [MSK] Prior orthopedic surgery
  - [MH] Depression or anxiety (diagnosed/treated)
  - [MH] Other psychiatric condition
  - [MH] Current psychotherapy
  - [OTH] Allergies requiring regular treatment
  - [OTH] Sleep apnea (diagnosed)
  - [OTH] Any condition requiring ongoing medication
- Include a preamble: "Select any conditions that apply. This information is used ONLY for this session to estimate PKV risk tier — it is never stored or persisted (GDPR Art. 9 compliant). Select none if no conditions apply."
