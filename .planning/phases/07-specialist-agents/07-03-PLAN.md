**`<process>` section with these phases:**

**Phase 1: Read Profile Data**
- Read `.finyx/profile.json`
- Extract: `investor.liquidAssets`, `investor.monthlyCommitments`, `investor.income.total`, `investor.marginalRate`
- Extract: `countries.germany.gross_income`, `countries.germany.marginal_rate`
- Extract: `countries.germany.brokers[].holdings[]` — for each holding: shares, cost_basis, asset_class
- Extract: `countries.brazil.brokers[].holdings[]` (if cross_border)
- Extract: `goals.primary_goals[]`, `goals.investment_horizon`, `goals.risk_tolerance`
- Extract: `pension.de_rentenpunkte`, `pension.br_inss_status`, `pension.target_retirement_age`

**Phase 2: Compute Net Worth Snapshot (PROJ-01)**
- **Assets calculation:**
  - Liquid assets: `investor.liquidAssets` (directly from profile)
  - Portfolio value (DE): sum of `shares x cost_basis` for each holding across all `countries.germany.brokers[]`
  - Portfolio value (BR): sum of `shares x cost_basis` for each holding across all `countries.brazil.brokers[]` (convert to EUR at stated assumption rate if needed, or keep in BRL with note)
  - Pension value (DE): `pension.de_rentenpunkte x current_point_value` — reference: 1 Rentenpunkt ~= EUR 39.32/month (2025 West). Present as monthly pension entitlement, not lump sum.
  - Total assets = liquid + DE portfolio + BR portfolio (note: pension shown separately as monthly entitlement)
- **Liabilities calculation:**
  - Profile has no explicit liabilities section
  - Use `investor.monthlyCommitments` as monthly obligation proxy — this is NOT a balance sheet liability
  - Note explicitly: "Profile does not contain balance sheet liabilities (mortgages, loans). Monthly commitments (EUR X/month) shown as recurring obligations."
  - If user has goals that imply debt (e.g., real estate with financing), note but do not fabricate amounts
- **Net worth = total assets - liabilities (if any explicit ones exist, otherwise net worth = total assets)**
- IMPORTANT: Use `cost_basis x shares` as portfolio value — this is a conservative floor. Flag as MEDIUM CONFIDENCE with note: "Portfolio valued at cost basis — actual market value may differ. Run /finyx:broker for current NAV."
- Round all figures to EUR 1,000

**Output table format:**
```
| Component | Amount | Confidence |
|-----------|--------|------------|
| Liquid Assets | EUR XX,000 | HIGH |
| DE Portfolio (cost basis) | EUR XX,000 | MEDIUM |

| **Total Assets** | **EUR XXX,000** | |
| Monthly Obligations | EUR X,XXX/month | HIGH |
| **Estimated Net Worth** | **EUR XXX,000** | MEDIUM |
```

**Phase 3: Compute Savings Rate**
- Net-after-mandatory income: use benchmarks.md Section 1 formula (same as allocation agent)
- Monthly savings capacity = net_monthly_income - monthlyCommitments
- Savings rate = monthly_savings / net_monthly_income x 100
- Compare against benchmarks.md Section 4 investment rate target
- This feeds into Phase 4 pace calculations

**Phase 4: Goal Pace Tracking (PROJ-02)**
- For each goal in `goals.primary_goals[]`:
  - Determine goal target amount (if specified in goal definition)
  - Determine current progress toward goal (from assets data)
  - Compute remaining amount = target - current_progress
  - Compute months to reach goal:
    - Conservative: remaining / monthly_savings (no investment return)
    - Base: remaining / (monthly_savings + monthly_savings x assumed_annual_return/12)
    - Use `goals.risk_tolerance` to select appropriate assumed return rate:
      - Conservative risk tolerance: 3% annual real return
      - Moderate: 5% annual real return
      - Aggressive: 7% annual real return
  - Output: "At current rate, [goal] reached in [X-Y] months ([base]-[conservative])"
- If no primary_goals defined: output "No goals defined in profile — run /finyx:profile to add investment goals"
- Include pension pace if `pension.target_retirement_age` is set:
  - Current Rentenpunkte vs estimated needed for target pension income
  - Simple projection: current_points + (years_to_retirement x estimated_annual_points)

**Phase 5: Format Output**
- Wrap entire result in `<projection_result>` tags (per D-01)
- Include net worth snapshot table
- Include savings rate indicator
- Include goal pace for each goal
- All EUR amounts rounded to nearest EUR 1,000
- State all assumptions explicitly in an "Assumptions" section:
  - Portfolio valued at cost basis (not market)
  - Monthly savings = net income - monthlyCommitments
  - Investment return assumptions by risk tolerance
  - Pension point value (2025 West Germany)
- Append confidence level and disclaimer reference
